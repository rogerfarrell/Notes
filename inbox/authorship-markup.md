A simple markup language for authorship (for normies).

Sections:

```
# section name

Some paragraphs

section name #
```

Inlines:

```
This is a paragraph with #inlinename an inline inlinename#.
Some more of the paragraph.
```

# An alternate syntax?

Sections:

```
section name >

Some paragraphs

< section name
```

Inlines:

```
This is a paragraph with inlinename> an inline <inlinename.

```
